# Project Halo Web

Marketing website for Project Halo — an intelligent asset financing solution by [Ramn](https://ramn.com).

Built with React, TypeScript, Tailwind CSS and Vite. Supports Swedish and English.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview  # preview production build locally
```

## Lint

```bash
npm run lint
```
